### Sintaxis para duraciones
A veces, un comando requiere un parámetro de `tiempo`, por ejemplo, el comando para iniciar sorteos.
Se pueden utilizar las siguientes unidades para definir una duración:
- `s` segundos
- `m` minutos
- `h` horas
- `d` días
- `w` semanas
- `y` años

No solo eso, también puedes combinar las unidades, por ejemplo `7d12h` equivaldría a una duración de 7 días y 12 horas.

<CodeGroup>
```css Sintaxis Correcta ✅
50s
10m
7d
12h5m
1w2d
1y2w20m
```

```css Sintaxis Incorrecta ❌
50 segundos
10minutos
7 days 1w
12horas 5 minutos
```
